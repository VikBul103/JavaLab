Слід створити абстрактний клас, похідний від класу, який представляє основну сутність.
Цей клас, повинен декларувати відповідні операції читання і запису і реалізувати
можливості демонстрації взаємодії з файлами різних форматів.

Слід також створити два похідних класи від класу.
Один клас повинен бути доповненим можливостями створення текстового файлу,
читання даних з цього текстового файлу та запису цих даних в інший файл після сортування.
Другий клас повинен реалізовувати створення бінарного файлу даних, читання даних з цього файлу даних
та запису цих даних в інший файл дних після сортування.

Окрім роботи з файлами повинно бути реалізоване виведення результатів у консольне вікно.
